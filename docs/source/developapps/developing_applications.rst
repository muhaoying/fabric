This topic covers how to develop a client application and smart contract to solve a
business requirement using Hyperledger Fabric. Using a real world **Commercial
Paper** application, you'll learn about all the concepts and tasks required to
accomplish this goal. We assume that the blockchain network is already available.
本文覆盖如何开发一个客户端应用和智能合约来解决一个商业需求，使用Hyperledger Fabric。使用一个真是世界的**商业票据**应用，你将学习到所有的概念和任务 需要来实现这个目标。我们假设区块链网络已经存在了。

The topic is designed for multiple audiences:
本文为一下读者所设计
*	Solution and application architect
*  解决方案和应用架构师
*	Client application developer
*  客户端应用开发者
* Smart contract developer
*  智能合约开发者
* Business professional
*  商业专家

You can chose to read the topic in order, or you can select individual sections
as appropriate. Individual topic sections are marked according to reader
relevance, so whether you're looking for business or technical information it'll
be clear when a topic is for you.
你可以顺序阅读本文，或者你可以根据需求适当选择单独章节阅读。每个章节都标注出适合的读者，所以无论你想找商业信息还是技术信息，都很清楚那些章节适合你。
